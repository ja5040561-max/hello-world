DatArk DB DR

PostgreSQL WAL 기반 실시간 DB 재해복구 모듈

PostgreSQL의 WAL(Write-Ahead Log)을 실시간으로 분산 클러스터에 스트리밍하여 RPO 4~10초, RTO 15분 이내를 달성하는 DB 전용 재해복구 모듈입니다. 16MB Chunk 분할, AES-256 Zero-Knowledge 암호화, CID 기반 무결성 검증으로 완벽한 데이터 보호를 보장합니다.

  • WAL 실시간 스트리밍: 서비스 무중단 상태에서 초 단위 변경 이력을 분산 클러스터에 전송 (RPO 4~10초)
  • pg_basebackup 비차단 스냅샷: DB 서비스 중단 없이 전체 백업 실행, 19.8배 압축 효율
  • PITR 시점 복구: 매니페스트 + 8채널 병렬 복구로 "어제 오후 3시 15분 10초" 상태로 정확히 복원
  • Cross-Host DR: 서울 서버 장애 시 부산 등 타지역 새 서버에 즈시 복구 (NHN Cloud 검증 완료)

DB DR 핵심 기능 상세

PostgreSQL WAL 기반의 실시간 복제와 시점 복구를 위한 기능 모음

WAL 실시간 스트리밍

PostgreSQL의 WAL을 실시간으로 분산 클러스터에 전송. 서비스 무중단 상태에서 RPO 4~10초 달성. Durable-Ack로 유실 0%.

pg_basebackup 비차단 스냅샷

DB 서비스 중단 없이 전체 백업 실행. 100GB 원본이 gzip 압축을 거쳐 5.23GiB로 저장 (19.8배 효율).

PITR 시점 복구

매니페스트 + CID 목록으로 특정 시점의 DB 상태를 정확히 재현. 8채널 병렬 복구로 RTO 15분 이내.

Cross-Host DR

CID 기반 위치 독립성으로 서울 센터 장애 시 부산 등 타지역 새 서버에 즉시 복구. NHN Cloud 검증 완료.

Zero-Knowledge 암호화

AES-256/ChaCha20으로 고객 서버에서 암호화 완료 후 전송. KONEKSI 서버에서도 데이터 내용 열람 불가.

자가치유 및 안전장치

Data Scrubbing + Re-pinning 자동 복원. Slot Pressure 3단계 경보로 디스크 풀 위험 자동 방지.
