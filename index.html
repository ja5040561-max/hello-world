데이터를 지키는 세 가지 핵심 기술

DatArk는 수학적 증명과 분산 아키텍처로 데이터의 기밀성, 무결성, 가용성을 동시에 보장합니다.

Zero-Knowledge 암호화

AES-256 / ChaCha20으로 고객 서버에서 암호화 완료 후 전송합니다. KONEKSI 서버에서도 데이터 내용을 열람할 수 없는 완전한 데이터 주권 구조입니다.

CID 기반 무결성 증명

SHA-256 해시로 생성된 CID(Content Identifier)가 곧 데이터의 주소입니다. 1비트라도 변조되면 CID가 완전히 달라져 위변조를 즉각 감지합니다.

자가치유 클러스터

클러스터 노드가 상호 데이터를 대조(Data Scrubbing)하여 손상을 사전 감지하고, 정상 노드의 복사본으로 자동 복원(Re-pinning)합니다. 99.999% 보존율.

Measured Performance

실측으로 증명된 성능

4~10초

RPO — 사실상 실시간

WAL 실시간 스트리밍으로 사고 발생 시 최대 4~10초 전 데이터까지 복구합니다. Durable-Ack 프로토콜로 유실 0%를 보장합니다.

15분 이내

RTO — 100GB 기준

8채널 병렬 다운로드와 매니페스트 기반 복구로 100GB 데이터를 15분 이내에 완전 복원합니다.

19.8배

압축 효율 — 94% 절감

gzip 압축과 중복 제거로 100GB 원본이 5.23GiB로 저장됩니다. 스토리지 비용을 최대 94% 절감합니다.

Zero-Touch Installation

기존 시스템 그대로, 6분이면 완료

DB를 재시작할 필요도, 서비스를 중단할 필요도 없습니다. 16MB 단일 실행 파일 하나로 즉시 보호가 시작됩니다.

01

바이너리 복사

16MB 단일 Go 실행 파일을 서버에 전송합니다. 외부 라이브러리 의존성이 없습니다.

약 1분
02

최소 권한 설정

백업 전용 사용자에 REPLICATION 권한만 부여합니다. 데이터 조회·수정은 원천 차단됩니다.

약 2분
03

설정 파일 작성

config.yaml에 클러스터 주소와 인증 토큰만 입력합니다. 메모장 수준의 편집으로 충분합니다.

약 2분
04

서비스 시작

시스템 서비스로 등록하면 즉시 보호가 시작됩니다. DB 재시작이 필요 없습니다.

약 1분
서비스 중단 0초 — DB 무중단 설치
현관문에 스마트 도어락만 다는 것처럼, 기존 인프라를 전혀 건드리지 않고 보안만 강화합니다. 설치 총 소요 시간 약 6분, 서비스 중단 시간 0초.
