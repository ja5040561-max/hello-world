왜 DatArk인가

기존 솔루션이 해결하지 못한 문제를 DatArk만의 방식으로 해결합니다. 가장 고도화된 기술력으로 비즈니스 연속성을 보장합니다.

Zero-Knowledge 데이터 주권

고객 서버에서 AES-256/ChaCha20 암호화를 완료한 뒤 전송합니다. KONEKSI 서버에 도착한 데이터는 이미 암호화 상태이므로 개발자나 관리자도 내용을 절대 열람할 수 없습니다.

vs 기존 솔루션: 서버 측 암호화 → 관리자가 원본 접근 가능

15분 이내 전체 복구

매니페스트 기반 6단계 복구와 8채널 병렬 다운로드로 100GB DB를 15분 이내에 완전 복원합니다. WAL Replay 약 30초로 최근 최적화 달성.

vs 기존 DR: 수동 복구 절차 → 평균 4~24시간 다운타임

법적 증빙 수준 무결성

분산 해시 트리 구조로 데이터의 원본성을 수학적으로 증명. 금융·공공 규제 감사 및 법적 분쟁에서 즉시 활용 가능한 무결성 보고서를 발행합니다.

vs 기존 솔루션: 체크섬만 제공 → 위변조 증명 불가

Cross-Host 재해복구

CID 기반 위치 독립적 데이터 관리로 서울 센터 장애 시 부산 등 타지역 새 서버에 즉시 복구. NHN Cloud 환경에서 Cross-Host 복구 테스트 완벽 성공.

vs 중앙집중식: 서버 장애 = 전체 서비스 중단

19.8배 스토리지 효율

gzip 압축과 Chunk 단위 중복 제거로 100GB 원본이 5.23GiB로 저장됩니다. Cloud + IDC 하이브리드 구조로 운영비 80% 절감이 가능합니다.

vs 전통 백업: 전체 파일 전송 → 과도한 대역폭 소모

엔터프라이즈 완전 통합

기존 인프라 교체 없이 에이전트 설치만으로 즉시 연동. DatArk Manager의 중앙 관제 대시보드로 전사 데이터 자산을 단일 화면에서 모니터링합니다.

vs 레거시 툴: 교체 비용 수억 원 + 수개월 마이그레이션

제품 비교

File DR vs DB DR

데이터 유형과 서비스 요구에 맞는 최적의 모듈을 선택하세요

항목 File DR DB DR
보호 대상 비정형 파일 (문서, 이미지, 영상, CAD 등) PostgreSQL DB (트랜잭션 데이터)
동기화 방식 블록 단위 증분 동기화 (Rsync) WAL 실시간 스트리밍 + pg_basebackup
RPO 목표 ≤ 1분 4~10초 (WAL 실시간)
RTO 목표 단일 파일 수초 / 전체 환경 의존 15분 이내 (100GB 기준)
주요 위협 방어 랜섬웨어, 파일 유실, 위변조 DB 장애, 휴먼 에러, 데이터센터 재해
배포 형태 클라우드형 / 온프레미스형 Cloud + IDC 하이브리드 / 온프레미스형
권장 대상 전 산업군 문서 중앙화 환경 금융, 공공, 미션 크리티컬 + 타지역 DR
